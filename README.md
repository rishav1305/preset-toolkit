# preset-toolkit

Claude Code plugin for Preset/Superset dashboard management.

## Install

```
/plugin marketplace add rishavchatterjee/preset-toolkit-marketplace
/plugin install preset-toolkit@preset-toolkit-marketplace
```

## Quick Start

```
mkdir my-dashboard && cd my-dashboard
/preset setup
```

## Prerequisites

- Python 3.8+
- `pip install preset-toolkit`
